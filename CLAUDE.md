# プロジェクト概要

匿名掲示板システムの管理者向けUIを構築する。

管理者はブラウザから以下の操作を行える：

- 板 / スレッド / 投稿の一覧表示
- 削除・編集
- 投稿の詳細情報の確認（APIレスポンス含む）

---

# 技術スタック

- Python / FastAPI
- Jinja2（サーバサイドレンダリング）
- httpx（API連携）
- Docker / Docker Compose

---

# 認証

- 管理者ユーザ名 / パスワード + Cloudflare Turnstileトークンの入力

---

# 方針

- 管理者向けのためUIはシンプル・機能重視
- 可読性を重視した構成にする
- 適切にファイルを分割する
- ログ出力を行う
- docs/api-docs のAPI仕様を参照して実装する

---

# その他

- ローカルで docker compose up により起動できるようにする
- 必要な設定は .env で管理する
